Features
========

.. include:: summary.rst

* :doc:`Blitz additionals <blitz_additionals>` - Generate additional fields in Blitz action snippets
* :doc:`Dialog <dialog>` - Respond to prompts during execution/configuration of commands
* :doc:`Device command aliases <device_cmd_aliases>` - Switch devices faster by auto-creating aliases for ``device`` command
* :doc:`Loop <loop>` - Create loops to simplify repetitive operations
* :doc:`Parameters <parameters>` - LAMP parameters for Blitz parameter markups
* :doc:`Variables <variables>` - LAMP variables for Blitz variable markups
* :doc:`Jinja <jinja>` - Jinja2 support
* :doc:`Tools <tools>` - Available tools that help in LAMP workflow
* :doc:`Console log files <console_log_files>` - Store console log files that records all operations on each device
