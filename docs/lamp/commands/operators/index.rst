Operator commands
====================

These commands *operate* on device to acheive a certain task
representing the basic Blitz actions and are primarily used to autogenerate
these Blitz actions snippets.

* :doc:`execute` - Execute commands; autogenerates *'execute'* Blitz snippet
* :doc:`configure` - Configure CLI; autogenerates *'configure'* Blitz snippet
* :doc:`parse` - Parse commands; autogenerates *'parse'* Blitz snippet
* :doc:`api` - Invoke Genie API's; autogenerates *'api'* Blitz snippet
* :doc:`replay` - Run generated Blitz testcases; autogenerates *'run_genie_sdk'* Blitz snippet
* :doc:`sleep` - Sleep; autogenerates *'sleep'* Blitz snippet
* :doc:`bash_console` - Execute bash console commands; autogenerates *'bash_console'* Blitz snippet
